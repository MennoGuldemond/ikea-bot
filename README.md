# Ikea bot

A node program that polls an Ikea api to get stock information on a product

## Getting started

1. run 'npm install'
1. Add a .env file to the root with the following properties:
   - BOT_EMAIL_SERVICE=...
   - BOT_EMAIL=...
   - BOT*EMAIL*...
   - PRODUCT_NUMBER=...
   - POLLING_INTERVAL=...
   - CLIENT_ID=...
   - MAIL_TO=...
1. run 'node bot.js' to start the bot.
